# sophisticated filter effects
----------

These demos are a proof of concept to demonstrate that SVG filter effects can serve as some form of "CSS for type", making it possible to create sophisticated display typography with pure web technologies.

## Basic Effects:

The directory "basic-effects" contains the building blocks that can be used to onstruct a sophisticated SVG filter.

## Sophisticated Effects:

Examples of sophisticated filter effects constructed by combining multiple modules.

The directory "SVG" contains filters applied to SVG text.

The directory "HTML" contains the same filters, this time  applied to HTML Text
